KW1281 / K-Line Protocol

VAG 1.9 / 2.0 TDI
DISPLAY

Golf IV, Golf V, Passat B5, Passat B6, Octavia I/II, Leon I/II — ASZ, BJB, BKD, ARL, BMM and related engines · Bosch EDC15P+ ECU

The first FastLifeBG product — built around the KW1281 diagnostic protocol that VAG TDI ECUs have used since the 1990s. Reads directly from the EDC15 ECU's own diagnostic groups: the same data VCDS sees, the same data your mapper uses. Per-cylinder injection balance was the feature that justified the whole product — we caught a cyl 1 injector at +5.16 mg/stroke, way beyond what VCDS can even display.

Display Pages
  • Boost actual vs target
    Auto-scaled to your tune. Deviation coloured: red = overboost sustained 2s, amber = underboost 5s. Buzzer on overboost.
  • VNT vane position %
    Watch your variable nozzle turbo working in real time. Groups 11 polled every cycle.
  • Coolant, IAT, Fuel temperature
    All with custom thresholds. IAT shows seconds since last update — essential for spray testing.
  • Injection quantity (mg/stroke)
    How hard your injectors are working. From group 2 alongside RPM and engine load.
  • Engine load % + RPM
    Always visible. Your two master reference numbers.
  • Live boost graph
    Scrolling waveform of boost pressure over time. See spool, peak, and drop in one view.
  • Per-cylinder injection balance
    Cylinders 1–4 with mg/stroke deviation. Thresholds: >±2.8mg = RED FAULT, >±2.0mg = AMBER. Real values — not capped at ±2.99 like VCDS. From group 13, formula 6×(b-128)/128.
  • Auto-switch on fault
    If any cylinder stays at FAULT level for more than 5 seconds, display auto-switches from Page 1 to Page 2.
  • Battery voltage
    Live alternator and battery health. From group 16. Warn <12.5V, crit <11.5V.
  • Fuel consumption L/100km
    Calculated from live speed (group 6) and fuel flow (group 15). Active when speed >5 km/h.
  • 0–100 km/h timer
    Auto-triggered at launch (5 km/h). Stops at real 100 km/h with 5% speedometer correction applied. Results held until reset.
  • 0–200 km/h + 100–200 km/h
    Rolling pull timer triggers at 105 km/h indicated. Same correction. Comparable to Dragy results.
  • Peak boost — session record
    Highest boost seen since last reset. Persists until you clear it manually.
  • P-codes with full descriptions
    Not just the code — full decoded text per fault. Static faults flash red, sporadic = amber. Up to 10 faults displayed. Refreshes every 60 seconds.
  • VAG hex codes alongside P-codes
    Both the OBD2 P-code and the VAG internal hex code shown side by side.
  • Clean when clean
    "NO FAULTS" displayed large in green when the ECU is clear. No guessing.

Poll rateGroups 2+11 every cycle (~250ms), group 13 every 10s

PT-CAN Passive Broadcast · 500kbps

BMW M57 / N57
DISPLAY

E60 520d–535d, E61 Touring, E90/E91/E92/E93 318d–335d, E87 118d–123d, E83 X3, E70 X5 · M57D25, M57D30, M57N2 bi-turbo, N57 · ZF 6HP automatic where fitted

The BMW display listens passively to the PT-CAN bus — no requests sent, no bus load added. Every piece of live data is broadcast continuously by the DDE and EGS: boost, rail pressure, oil temperature, transmission fluid temperature, gear, torque. Rail pressure actual vs target is the defining feature — catch CP3 pump wear before it destroys your injectors. Plus a hardware GPIO relay that physically disconnects the MCP2515 from CAN during ECU remapping.

Display Pages
  • Boost actual vs target
    Both from CAN ID 0x1D0. Auto-scaled to your remap. Deviation coloured.
  • Rail pressure actual vs target
    From CAN ID 0x3B4. Deviation alerts: >200 bar for 2s = amber, >400 bar for 2s = red + beep. Pump failure early warning.
  • Oil temperature
    M57 has a proper oil temp sensor — confirmed on CAN ID 0x1B6. Not just a switch.
  • Coolant + IAT
    Both from 0x1D0 alongside boost data. Custom thresholds matched to your engine.
  • Gear position + trans fluid temp
    From 0x0BA — ZF 6HP has a fluid temp sensor readable via PT-CAN. Warn when fluid creeps up.
  • Engine torque Nm
    Actual torque from 0x0B6. Wheel torque calculable from DDE + EGS torque × gear ratio.
  • Rail pressure live graph
    Scrolling waveform showing pump health trend over time. Intermittent drops become visible before they become failures.
  • Fuel consumption L/h
    From 0x3B4 — same message as rail pressure. Direct from DDE.
  • Battery voltage
    Live alternator health monitoring.
  • Oil pressure status
    M57 factory sensor is a switch, not analog — shows OK / Warn / Critical. Optional analog upgrade available for bar values.
  • 0–100, 0–200, 100–200 timers
    Speed from DSC wheel sensors (0x19E — four individual wheel speeds for maximum accuracy). Configurable speedometer correction.
  • Peak boost — session
    Highest boost recorded, held until reset.
  • Gear position + selector state
    P/R/N/D/S/M from 0x0BA and 0x1D2. Always visible.
  • Transmission fluid temp — live
    Colour-coded: green = safe, amber = warm, red = hot. High ATF temp is the number one cause of ZF 6HP failure on tuned cars.
  • TCC lockup %
    Torque converter lockup from 0x1A2. See exactly when the converter locks and slips.
  • Boost graph vs RPM Coming
    Live X/Y graph of boost vs RPM drawn in real time as you drive. See exactly where your turbo spools.
  • Shift quality graph Coming
    Hydraulic pressure during each gear change. Identify soft shifts, hard shifts, and slipping.
  • Engine P-codes (0x7E8)
    OBD2 active request sent every 60s. Same MCP2515, same CAN wires — zero extra hardware. Mode switching takes ~150ms.
  • Transmission faults (0x7E9)
    EGS fault codes alongside engine faults. Covers boost, injector, sensor, glow plug, EGR, DPF, and transmission codes.
  • Brightness slider
    Touch control on the ILI9341 screen. Three levels via PWM on backlight pin.
  • Colour theme selector
    Switch between configured themes on-screen.
  • Rail pressure max scale
    Set to match your sensor: stock 1600 bar, R90 CP3 builds up to 2000 bar, high power up to 2700 bar.
  • Diagnostic Mode — hardware relay
    GPIO-controlled relay physically disconnects the MCP2515 from PT-CAN. Zero electrical presence on the bus during ECU remapping. Safe for flashing. No competitor implements proper hardware isolation.

CAN FD + UDS Active Polling · MCP2518FD

VW GOLF MK8 R
DISPLAY

Golf R MK8, Golf GTI MK8, Audi S3 8Y, Cupra Formentor, Skoda Octavia Mk4 · EA888 Gen 4 (EVO4) · SIMOS 19.6 ECU · DQ381 DSG · MQB Evo platform

The most technically complex FastLifeBG product. The MK8 R uses CAN FD — not classic CAN — so the hardware is completely different (MCP2518FD, not MCP2515). All live data is obtained via UDS active polling over the OBD2 port: we request each parameter individually, the ECU responds. SFD does not block read-only data access — no tokens, no subscriptions, no VW server authentication required. We just read.

Display Pages
  • Boost actual vs target
    Polled via UDS 0x22 service. Auto-scaled to your tune. Deviation coloured. PID confirmed from MK7 community research, verified on MK8.
  • RPM + throttle position
    RPM via UDS PID 0x22D1. Throttle via standard OBD2 mode 0x01. Polled at 100ms.
  • Coolant + IAT + oil temperature
    Coolant and IAT via standard OBD2. Oil temp via UDS. All with custom colour thresholds.
  • Engine load % + fuel trim
    Load and fuel trim from standard OBD2 PIDs. Fuel trim deviation is an early knock/overrun indicator.
  • Gear position (P/R/N/1–7)
    Via UDS PIDs 0x3816 (engaged gear) and 0x3815 (selector). Confirmed on MK7, same on MK8.
  • Live boost graph
    Scrolling boost waveform. Coming: boost vs RPM X/Y graph drawn in real time.
  • 0–100, 0–200, 100–200 timers
    Speed via standard OBD2 PID 0x0D. Launch trigger at 5 km/h. 5% correction default.
  • Peak boost — session
    Highest boost recorded this session. Held until reset.
  • DSG fluid temperature
    Warn >110°C, critical >130°C. High DSG fluid temp is the main cause of DQ381 failures on tuned cars. See it before it becomes a repair bill.
  • DSG gear + shift indicator
    Gear position from UDS. Visual shift indicator configurable to your power band.
  • Battery voltage
    Live charging system health.
  • DSG shift quality graph Coming
    Real-time hydraulic pressure curve during each gear change. Same data transmission tuners use.
  • Engine P-codes
    UDS mode 0x03 request every 60s. ISO-TP multi-frame handling. Boost, sensor, injector, lambda codes.
  • DSG fault codes
    Transmission ECU faults alongside engine faults. Same clean layout.
